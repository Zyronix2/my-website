the web:


The Web (or World Wide Web) is a collection of documents and other files.
Most of the documents are made using Hypertext Markup Language (HTML).

What is HTML?


HTML stands for Hypertext Markup Language. It's used to define the structure or layout
of a web page.

key terms:


  • Markup: is a text that has a spacial meaning

  • syntax: are sets of rules having to do with how the parts of a language can be put together.
  • nesting: The idea that some bits of code can go inside other bits of code.

  • developers often use the word whitespace as a general term for spaces, tabs, and line breaks.

  • Void elements: are html elements that have no contents or no closing tags an example can be: elaments likebr with the angled brackets

  • Class: A class is a group of things with the same characteristics.
    To create a class, you need to use the attribute "class" in the HTML opening tag.
    Then in your CSS, you use the same name you gave the class attribute in your HTML, except you place a . in front of it.
